# Workflow Examples

## Full Pipeline

Use when the user wants the system to decide the right path:

```text
Use $syft-news-pipeline to build my profile from Syft CLI and then produce today's personalized daily briefing.
```

## Profile First

```text
Use $syft-profile-summary to read syft following and generate following_topics.md, raw_interest_block.txt, and profile_summary.md.
```

## Daily Edition

```text
Use $syft-daily-briefing to generate today's personalized daily briefing from the current profile artifacts.
```

## Storyline Tree

```text
Use $syft-storyline-tree to turn today's briefing into a trunk-branch storyline tree in markdown, HTML, and JSON.
```

## Backfill

```text
Use $syft-storyline-backfill to deepen the EVA branch over the last 30 days without rebuilding the whole tree.
```

## Durable Guidance

```text
Use $syft-guidance-rulebook to preserve my long-term preference that camera, EVA, and onsen lines should not be collapsed into one generic side-interest section.
```
